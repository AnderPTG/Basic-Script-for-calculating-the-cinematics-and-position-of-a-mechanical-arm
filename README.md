# Basic-Script-for-calculating-the-cinematics-and-position-of-a-mechanical-arm
In this Script you will find some basic equations in which I used for calculating the position of an mechanical arm in matlab

for this script I based myself on the positioning of the arm using basic cinematics
$H = C^2 + C^2$ in which the different cathetus will determine the movement of the arm
also I used trigonometry for calculating the angle of inclination for in the end having the cinematics of the arm in the end I used law of cosins for knowing the final position in whic I want the arm to be positioned which is 
$θ = (a^2+b^2-c)/(2ab)$ 
in which in this case a = r, b = L, c = recorrido
